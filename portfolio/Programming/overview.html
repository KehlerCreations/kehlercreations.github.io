Destructible environment for Two Dollar Suits

Everything in Two Dollar Suits is destructible. I have developed my own system for detecting holes in walls to split colliders, leaving a gap in-between for players and NPCs to walk through. Pathfinding is updated to reflect the new available space.

Advanced weather system for Walking With The Wild

Walking With The Wild features dynamic rain, wind and fog. I have built a system to change these values independently. Wind effects change both rain direction and material attributes for vertex displacement. Thick fog cripples vision and encourages players to use their other senses to survive.

I made a custom vertex displacement shader in Unity's shader graph to handle grass and tree deformation.

Creature senses for Walking With The Wild

Creatures in Walking With The Wild feature their own sense components. All creatures emit sound, scent, and make vibrations in the ground. The strength of these emissions are driven by variables. Creatures also differ in how sensitive they are to picking up such emissions from others.

The Spider Branch has poor eyesight, but has acute hearing, encouraging players to find alternative paths or sneak while close to this creature.

Simulated limbs for The While Hole System

The White Hole System features a limb simulation system I designed and built completely from scratch. Both the player and NPCs interface with this system move around the map. The game also features predictive foot movement for smoother traversal up and down stairs. Due to shifting priorities, the system never got quite stable enough for a public release.

Parkour pathfinding for The While Hole System

Since the game is built in GameMaker, which only features rudimentary built-in pathfinding options, I have integrated my own A* algorithm to enable navigation in this sidescroller parkour world. NPCs intelligently navigate from point to point, using appropriate behavior as needed, such as jumping, crouching or swinging.

Pathfinding nodes are automatically connected based on proximity and direct paths. Nodes can be connected either one-way or two-way. For example, a rooftop node may be connected to a floor node for NPCs to jump down, but they would not attempt to jump up to the rooftop from the ground.

Custom HLSL fragment shader for adjustable 2D shading

The parallax backdrop needs to adapt to lighting conditions, so I have written my own system using simple HLSL and color maps to make this possible.

The lighting is driven by three maps. The base map is a simple color as seen on the left. Highlights are driven by the center map, which tells the engine what to highlight based on time of day. The right map selects which lights in building windows are turned on at any given time.
